# 发布流程

## 创建release分支
>* git flow release start 'version num'

## 结束release分支
>* git flow release finish 'version num'

## dev流更新到远程仓库
>* git push origin develop

## master流更新到远程仓库
>* git push origin master

## 发布版本号
>* git push --tags

## hotfix分支创建

>* git flow hotfix start <release>
>* git flow hotfix finish <release>
